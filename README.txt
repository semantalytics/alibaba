Alibaba is a Create Read Update and Delete Entity Browser with
configurable views and search patterns.
 
Documentation is included under the doc/ directory or on the website
http://www.openrdf.org/
 
Maven 2.x is used as the build script it can be downloaded at
http://maven.apache.org/
Running "mvn install" will download the dependencies and build the jar files.
